# 🥾 Fiche Sécurité RCY

Application personnelle de sécurité destinée aux adhérents du Rando Club Yerrois.

## Objectif

Permettre à chaque adhérent de conserver sur son téléphone :

- ses contacts d'urgence ;
- ses informations médicales essentielles ;
- son numéro de licence ;
- un QR Code d'urgence.

## Principes

- Données conservées uniquement sur le téléphone.
- Aucune donnée stockée par le club.
- Fonctionnement hors connexion.
- Installation sur Android et iPhone.
- Gratuit pour tous les adhérents.
